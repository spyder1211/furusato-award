## 🚀 クイックスタート

### 初めて開発を始める場合

1. [setup.md](./setup.md) を読んで環境をセットアップ
2. [progress.md](./progress.md) で現在の進捗を確認
3. [CLAUDE.md](../CLAUDE.md) で技術仕様を確認
4. 開発開始！

### 既存環境で開発を再開する場合

1. [progress.md](./progress.md) で進捗状況と次のタスクを確認
2. 必要に応じて [setup.md](./setup.md) のコマンドでキャッシュクリア等を実行
3. 開発再開！
